WHAT IS THIS?
=============

An AI-powered system that automatically analyzes your Airflow DAGs and
recommends optimizations using 4+ integrated tools:

  1. Claude AI           → Intelligent analysis & recommendations
  2. Airflow REST API    → Real DAG metrics (duration, frequency, errors)
  3. Prometheus Metrics  → Performance data (CPU, memory, execution time)
  4. Chroma Vector DB    → Pattern learning (learns from past optimizations)

QUICK START (5 minutes)
======================

1. Setup environment:
   cd /Users/nitinkamal/code/Spark/thedatamonk-analytics/airflow-optimization-agent
   python -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt

2. Configure credentials:
   cp .env.example .env
   nano .env  # Add your API keys

3. Run analysis:
   python -m src.cli analyze-all

4. View results:
   • optimization_report.json    (full report)
   • Console output              (summary)
