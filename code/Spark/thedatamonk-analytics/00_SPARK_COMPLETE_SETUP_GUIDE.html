⚙️ Apache Spark Complete Setup Guide

Step-by-Step Installation & Project Structure for macOS

📋 Installation Overview

Here's what you'll install and why:

Component Version Size Purpose Installation Time
Homebrew Latest ~100MB Package manager for macOS 2-3 min
Java JDK 11 11.0+ ~300MB Spark runtime (required) 3-5 min
Apache Spark 3.5.0+ ~180MB Distributed compute engine 2-3 min
Python 3 3.9+ ~80MB Python interpreter (PySpark) Already on Mac
pip packages Latest ~500MB Jupyter, pandas, numpy, etc. 5-10 min
Total Installation Time: ~15-25 minutes
Homebrew → Java → Spark → Python packages → Project setup
1
Install Homebrew (macOS Package Manager)

What: Homebrew makes installing programs easy on Mac

Check if already installed:

brew --version # If installed, you'll see version number # If not, you'll get: command not found

If NOT installed, run this:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)" # This will: # 1. Download Homebrew # 2. Install it to /usr/local/opt (on Intel Mac) # 3. Setup environment variables # Takes 2-3 minutes

Verify installation:

brew --version # Should output: Homebrew 4.x.x
💡 What Homebrew Does

Homebrew = App Store for Mac developers. Instead of downloading installers, you use:

brew install java # Instead of manual download brew install apache-spark # Simple one-line install
2
Install Java JDK (Spark's Runtime)

What: Spark runs on Java Virtual Machine (JVM). Java JDK is required.

Check if already installed:

java -version # If installed: shows version (e.g., 11.0.20) # If not: command not found

Install Java JDK 11:

brew install openjdk@11 # Downloads and installs Java 11 # Takes 3-5 minutes

Set JAVA_HOME environment variable:

This tells Spark where to find Java. Add to your shell config file:

# Open shell config file (for zsh): nano ~/.zprofile # Add these lines at the end: export JAVA_HOME=$(/usr/libexec/java_home -v 11) export PATH=$JAVA_HOME/bin:$PATH # Save: Ctrl+O, Enter, Ctrl+X

Apply changes:

source ~/.zprofile

Verify Java installation:

java -version echo $JAVA_HOME # Should show Java version and /path/to/java
⚠️ Common Issue: "command not found"

If you still can't find Java after installation:

# Restart terminal completely (close and reopen) # Then try again java -version
3
Install Apache Spark

What: The actual Spark framework. Will be installed at /usr/local/opt/apache-spark

Install Spark:

brew install apache-spark # Downloads and installs Spark # Takes 2-3 minutes

Set SPARK_HOME environment variable:

# Edit ~/.zprofile again nano ~/.zprofile # Add these lines: export SPARK_HOME=/usr/local/opt/apache-spark/libexec export PATH=$SPARK_HOME/bin:$PATH # Save: Ctrl+O, Enter, Ctrl+X

Apply changes:

source ~/.zprofile

Verify Spark installation:

spark-shell --version echo $SPARK_HOME # Should show Spark version and path
✅ If you see version numbers, Spark is installed!
4
Install Python Packages for Data Science

What: Libraries for working with Spark: Jupyter (interactive notebooks), pandas, numpy, etc.

Check Python version:

python3 --version # Should show 3.9 or higher

Create requirements file:

cd ~/code/spark cat > requirements.txt << 'EOF' jupyter==1.0.0 jupyterlab==4.0.0 pandas==2.0.3 numpy==1.24.3 matplotlib==3.7.1 scipy==1.11.1 scikit-learn==1.3.0 pyspark==3.5.0 EOF

Install packages:

pip3 install -r requirements.txt # Takes 5-10 minutes depending on internet speed

Verify installations:

python3 -c "import pyspark; print(pyspark.__version__)" jupyter --version # Both should show version numbers
💡 What Each Package Does
  • jupyter/jupyterlab: Interactive notebook environment (like Google Colab)
  • pandas: Data manipulation library
  • numpy: Numerical computing library
  • matplotlib: Plotting and visualization
  • scipy: Scientific computing
  • scikit-learn: Machine learning library
  • pyspark: Python API for Spark (already with Spark, but pip version keeps it updated)
5
Create Project Directory Structure

What: Organize your learning materials in a clean folder structure

Create directory structure:

mkdir -p ~/code/spark/phase-1-spark-internals/{week-1,week-2,week-3,week-4} mkdir -p ~/code/spark/phase-2-streaming/{week-5,week-6,week-7,week-8} mkdir -p ~/code/spark/phase-3-system-design/{week-9,week-10,week-11,week-12} # Create subdirectories for each week for week in ~/code/spark/phase-*/week-*; do mkdir -p "$week/{data,scripts,notebooks,output,config}" done

Your folder structure will look like:

~/code/spark/
phase-1-spark-internals/
week-1/
data/ ← Put CSV/JSON files here scripts/ ← Python code notebooks/ ← Jupyter notebooks output/ ← Results/screenshots config/ ← Config files
week-2/ (same structure) week-3/ (same structure) week-4/ (same structure)
phase-2-streaming/ (same pattern) phase-3-system-design/ (same pattern) requirements.txt ← Installed earlier README.md ← Project overview

Create a README for your project:

cat > ~/code/spark/README.md << 'EOF' # Spark Learning Project - 12 Week Intensive This project contains all materials for the 12-week L5 Data Engineer intensive. ## Project Structure - **phase-1-spark-internals/**: Weeks 1-4 (Spark execution, optimization) - **phase-2-streaming/**: Weeks 5-8 (Streaming, Kafka, distributed systems) - **phase-3-system-design/**: Weeks 9-12 (System design, operations) ## How to Run Code ```bash cd ~/code/spark/phase-1-spark-internals/week-1/scripts python3 01_lazy_evaluation.py ``` ## How to Use Jupyter ```bash cd ~/code/spark jupyter lab # Opens http://localhost:8888 in browser ``` ## Environment Variables Java and Spark paths are set in ~/.zprofile Check with: ```bash echo $JAVA_HOME echo $SPARK_HOME ``` ## Quick Links - Spark UI: http://localhost:4040 (while Spark job is running) - Jupyter: http://localhost:8888 (after `jupyter lab` command) EOF

Create Sample Data & Config Files

What: Sample CSV data and configuration files to get started

Create sample data:

cat > ~/code/spark/phase-1-spark-internals/week-1/data/sample.csv << 'EOF' id,name,age,city,salary 1,Alice,28,NYC,75000 2,Bob,34,NYC,85000 3,Charlie,26,LA,65000 4,David,29,LA,72000 5,Eve,31,SF,95000 6,Frank,25,SF,60000 7,Grace,32,NYC,88000 8,Henry,27,LA,68000 9,Ivy,30,SF,92000 10,Jack,35,NYC,98000 EOF

Create Spark config file:

cat > ~/code/spark/phase-1-spark-internals/week-1/config/log4j.properties << 'EOF' log4j.rootCategory=WARN, console log4j.appender.console=org.apache.log4j.ConsoleAppender log4j.appender.console.target=System.err log4j.appender.console.layout=org.apache.log4j.PatternLayout log4j.appender.console.layout.ConversionPattern=%d{yy/MM/dd HH:mm:ss} %p %c{1}: %m%n # Set specific logger levels for debugging log4j.logger.org.apache.spark.sql=INFO log4j.logger.org.apache.spark.scheduler=INFO EOF

Create a template Python script:

cat > ~/code/spark/phase-1-spark-internals/week-1/scripts/00_template.py << 'EOF' #!/usr/bin/env python3 """ Template Spark Script Week 1: Spark Execution Model """ from pyspark.sql import SparkSession import sys def main(): # Create Spark Session spark = SparkSession.builder \ .appName("WeekX-Topic") \ .config("spark.sql.adaptive.enabled", "true") \ .getOrCreate() print(f"Spark Version: {spark.version}") print(f"Python Version: {sys.version}") # Read data df = spark.read.csv( "../data/sample.csv", header=True, inferSchema=True ) print("Data shape: {} rows".format(df.count())) df.show() # Stop Spark spark.stop() if __name__ == "__main__": main() EOF chmod +x ~/code/spark/phase-1-spark-internals/week-1/scripts/00_template.py

8
Run Your First Spark Job

What: Execute a simple Spark program to confirm everything works

Create first job:

cat > ~/code/spark/phase-1-spark-internals/week-1/scripts/01_first_job.py << 'EOF' #!/usr/bin/env python3 """ First Spark Job - Verify installation """ from pyspark.sql import SparkSession # Create session spark = SparkSession.builder \ .appName("FirstJob") \ .getOrCreate() print("=" * 50) print("✅ Spark Session Created Successfully!") print("=" * 50) # Read CSV df = spark.read.csv( "../data/sample.csv", header=True, inferSchema=True ) print("\n📊 Data loaded:") df.show() print("\n📈 Statistics:") df.select("age", "salary").describe().show() print("\n🎯 Filtering age > 28:") df.filter(df.age > 28).show() spark.stop() print("\n✅ Job completed!") EOF

Run the job:

cd ~/code/spark/phase-1-spark-internals/week-1/scripts python3 01_first_job.py

Expected output:

================================================== ✅ Spark Session Created Successfully! ================================================== 📊 Data loaded: +---+-------+---+----+------+ | id| name|age|city|salary| +---+-------+---+----+------+ | 1| Alice| 28| NYC| 75000| | 2| Bob| 34| NYC| 85000| ... ✅ Job completed!
💡 What Just Happened
  • Your Python script created a Spark Session (driver process)
  • Spark read the CSV file (RDD created)
  • Transformations were lazy (just planning)
  • When you called .show(), execution happened (action)
  • Results printed to console
9
Access Spark UI Dashboard

What: Spark UI shows job execution visualization (critical for learning)

While your job is running (or just after), open browser:

http://localhost:4040

You'll see:

  • Jobs tab: List of jobs and their status
  • DAG Visualization: Your execution plan visualized
  • Stages: Breakdown of stages and tasks
  • Executors: How much memory each executor used
  • Storage: Cached RDDs in memory
⚠️ Spark UI Tips

UI is only available while Spark context is alive (job running or just finished). Give yourself 30 seconds to screenshot before it disappears!

10

What: Interactive notebook environment (optional but useful for exploration)

Launch Jupyter:

cd ~/code/spark jupyter lab

Expected output:

Open browser and create new Python notebook:

# In Jupyter cell, run: from pyspark.sql import SparkSession spark = SparkSession.builder \ .appName("Jupyter") \ .getOrCreate() df = spark.read.csv("phase-1-spark-internals/week-1/data/sample.csv", header=True) df.show()
💡 Jupyter vs Scripts
  • Jupyter: Good for exploration, visualization, interactive learning
  • Scripts: Good for reproducible code, production jobs, version control
For this 12-week program, use both!

🔧 Troubleshooting Common Issues

Error Cause Solution
command not found: java Java not installed or PATH not set Run brew install openjdk@11 and add JAVA_HOME to ~/.zprofile
command not found: spark-shell Spark not installed or PATH not set Run brew install apache-spark and add SPARK_HOME to ~/.zprofile
JAVA_HOME is not set Environment variable not configured Add export JAVA_HOME=$(/usr/libexec/java_home -v 11) to ~/.zprofile
ModuleNotFoundError: pyspark PySpark not installed Run pip3 install pyspark
Spark UI not showing Job finished before opening UI Open http://localhost:4040 while job is running
Address already in use: 4040 Another Spark job already running Close previous Spark job or kill the process
